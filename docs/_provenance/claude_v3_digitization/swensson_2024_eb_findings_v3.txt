CORE CONSERVATION EQUATION
---------------------------------------------------------------------------
Each Set 1 intervention session contains 5 trials:

    I_t + K_t + S_t = 5

where:
    I_t = independent IDKPTM Set 1 responses
    K_t = independent correct answers
    S_t = slack/residual trials with neither independent IDKPTM nor
          independent correct answering
